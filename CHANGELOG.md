# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-26

### Added
- 初始版本发布
- 笔记管理功能（创建、编辑、删除）
- Markdown 编辑器支持实时预览
- 标签系统（添加、筛选）
- 文件夹层级管理
- 全文搜索功能
- 回收站（恢复、永久删除）
- 数据导入/导出（JSON格式）
- 暗黑模式支持
- 自动保存功能
- 键盘快捷键（Ctrl+N, Ctrl+F）
- 代码高亮（Prism.js）

### Technical
- React 18 + TypeScript
- Vite 构建工具
- Tailwind CSS 样式
- Zustand 状态管理
- LocalStorage 数据持久化
- OpenSpec 规范驱动开发
